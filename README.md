# ☀️ Weather Forecast App

A simple web app that shows current weather and 5-day forecast using OpenWeather API.

## 🔧 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## ✨ Features
- Shows temperature, humidity, and weather description
- Current location and 5-day forecast
- Responsive layout

## 🚀 Setup Instructions
1. Clone the repo
2. Open `index.html` in your browser

## 📌 API Used
- [OpenWeatherMap API](https://openweathermap.org/api)
